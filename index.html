자동으로 이동하지 않으면 아래 링크를 선택하세요:

내부망 접속 | 외부망 접속

문제가 지속되면 다음을 확인해 보세요:

  • 브라우저 캐시/히스토리를 삭제하고 다시 접속해 보세요.
  • 내부망에서 http://172.30.1.100/ping.png 에 직접 접속되는지 확인하세요.
  • 브라우저 개발자 도구(F12) → Console 탭에서 [AutoRedirect] 로그를 확인하세요.
  • 내부 서버(172.30.1.100)의 CORS 설정을 확인하세요. Apache의 경우 .htaccess에 Header set Access-Control-Allow-Origin "*"를 추가해야 할 수 있습니다.
